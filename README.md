# APK Dumper
Tool to extract & Join APKs from device or emulator (Nox Player) to desktop directories.
Work in Windows & Linux


- Download

```bash
python3 apkdumper.py -p <app> -o <output_dir>
```

- List all packages

```bash
python3 apkdumper.py
```
<br>

- Search

```bash
python3 apkdumper.py -s <search>
```
